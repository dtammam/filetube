Backup & Restore

Downloads a snapshot of this FileTube's configuration and everyone's personal state - folders, settings, watch progress, likes, reading positions, pins, view counts, and user accounts. Media files are never included.

Download backup This file contains every account's password hash. Treat it like a password: store it somewhere private, never share it.
Restoring replaces this FileTube's configuration and accounts with the backup's contents. A backup that does not include your own admin account is refused, so a restore can never lock you out.
