# Bundled fonts

## Roboto (`roboto.woff2`)

The `latin`-subset **variable** build of Roboto (weight axis 100–900), self-hosted
so FileTube's UI matches YouTube's typography while staying fully offline — no
Google Fonts CDN call at runtime.
