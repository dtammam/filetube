# Bundled icons

## Material Symbols (Outlined)

Twelve individual SVG icons from Google's Material Symbols set (Outlined
style), self-hosted so FileTube's chrome iconography (navigation, theme
toggle, rescan, folder reorder) stays fully offline — no Google Fonts/Icons
CDN call at runtime. Each SVG is used as a CSS `mask-image` and painted with
`currentColor`, so a single unmodified asset renders correctly in every
FileTube theme.
